# KingFisher BDD Automation Project

This project contains **BDD (Behavior Driven Development)** test automation scripts using **Cucumber, Selenium, TestNG/JUnit, and Java**.  
The main objective is to validate core functionalities through feature files written in Gherkin syntax.

---

##  Project Structure

KingFisher/
├── src/test/java/stepDefinitions/ # Glue code (step definitions)
├── src/test/java/runners/ # Test runner classes
├── src/test/resources/features/ # Cucumber feature files
├── src/test/resources/config/ # Config files (if any)
├── pom.xml # Maven dependencies
└── README.md # Project documentation


---

## ✨ Features Covered

- Add items to cart  
- Validate product prices  
- Calculate cart totals  
- Checkout flow
- Verify correct promocode
- Verify wrong promocode
